# 💕 Valentine's Day — For Carmen

A small, fun Valentine's Day site you can host on GitHub Pages.

## What’s in it

- **Personal header** — “Happy Valentine’s Day, Carmen”
- **Interactive card** — Click the envelope to open it and read the message
- **Click hearts** — Click anywhere on the page to spawn little hearts
- **Simple layout** — Works on phone and desktop

## Deploy to your GitHub site

1. **Create a new repo** (or use an existing one).
   - For your **user site** (e.g. `yourusername.github.io`): create a repo named `yourusername.github.io` and use the **root** of the repo for these files.
   - For a **project site** (e.g. `yourusername.github.io/valentine`): create any repo (e.g. `valentine`) and use the **root** or a **`docs`** folder (see step 3).

2. **Put the site files in the repo**
   - Copy into the repo:
     - `index.html`
     - `styles.css`
     - `script.js`
   - Commit and push.

3. **Turn on GitHub Pages**
   - Repo → **Settings** → **Pages**.
   - Under **Source**, choose **Deploy from a branch**.
   - Branch: **main** (or **master**).
   - Folder: **/ (root)** or **/docs** if you put the files in `docs`.
   - Save.

4. **Open the site**
   - User site: `https://yourusername.github.io`
   - Project site: `https://yourusername.github.io/valentine` (or whatever the repo name is).  
   It can take 1–2 minutes to appear the first time.

No build step or server needed — it’s just HTML, CSS, and JS.

## Edit the message

Open `index.html` and change the text inside the `.card-message` section (the part that starts with “To my favorite person,”) to your own message for Carmen.
